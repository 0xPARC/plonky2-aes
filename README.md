# plonky2-crypto-gadgets

> Warning: this code is unaudited.

- [aes-gcm](/aes-gcm): Plonky2 circuits for AES & AES-GCM encryption system as described at
	- AES: https://nvlpubs.nist.gov/nistpubs/FIPS/NIST.FIPS.197-upd1.pdf
	- AES-GCM (Galois Counter Mode): https://nvlpubs.nist.gov/nistpubs/Legacy/SP/nistspecialpublication800-38d.pdf
- [feistel](/feistel): Plonky2 library for a finite field analogue of the Feistel cipher as described [here](https://en.wikipedia.org/wiki/Feistel_cipher).
- [poseidon-cipher](/poseidon-cipher): Plonky2 library & circuits for Poseidon encryption as described [here](https://drive.google.com/file/d/1EVrP3DzoGbmzkRmYnyEDcIQcXVU7GlOd/view).
